# MindFlow — Mental Health Mood Tracker

> Understand your mind. Improve your life.

MindFlow is a modern Mental Health & Mood Tracking web application with AI-powered insights, emergency support, and daily wellness tools.

## Tech Stack
- **Frontend:** HTML5, CSS3, Vanilla JavaScript
- **Backend:** Python (Flask)
- **Database:** MySQL

## Features
- Authentication (Login, Register, Onboarding)
- Daily Mood Check-in & Journal
- AI Recommendations & Insights
- Reports & Analytics
- AI Chat Assistant
- Emergency Alert System
- Admin Panel

## Project Structure
```
mindflow/
├── frontend/        # HTML, CSS, JS
├── backend/         # Python Flask API
├── database/        # MySQL schema & migrations
├── static/          # Images, fonts, uploads
├── tests/           # Pytest test suite
└── docs/            # API & setup docs
```
