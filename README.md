# Mental_Health_mood_tracker
# 🧠 Mental Health Mood Tracker

An AI-powered Mental Health Mood Tracker web application designed to help users monitor emotional well-being, track mood patterns, receive AI-generated insights, and activate emergency support alerts when critical emotional risks are detected.

---

# 📌 Project Overview

Mental Health Mood Tracker is a smart wellness platform that allows users to:

- Track daily moods and emotions
- Write personal journal entries
- Monitor mental health progress
- Receive AI-powered emotional analysis
- View weekly and monthly reports
- Get personalized wellness recommendations
- Send emergency alerts to trusted contacts during emotional crisis situations

This project focuses on improving mental health awareness through intelligent monitoring and supportive intervention systems.

# 🚨 Emergency Alert Feature

One of the core features of this project is the Emergency Mental Health Alert System.

When the AI detects:
- Emotional breakdown
- High stress patterns
- Dangerous mental condition
- Risk behavior

The system automatically:
- Sends alerts to trusted contacts
- Notifies guardians/family members
- Activates emergency monitoring

This helps provide real-time emotional support and safety assistance.
